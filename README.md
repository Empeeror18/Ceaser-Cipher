# Caesar Cipher

A simple Python Caesar cipher program that can encrypt and decrypt text from the command line.

## Features

- Encrypt text using a Caesar cipher shift
- Decrypt text using a Caesar cipher shift
- Re-prompts for the shift value when invalid input is entered
- Simple interactive terminal interface

## Requirements

- Python 3.8 or newer

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Empeeror18/Ceaser-Cipher.git
cd Ceaser-Cipher
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the program with:

```bash
python main.py
```

## Example

```text
Do you want to encrypt or decrypt? (e/d): e
ENCRYPTION MODE SELECTED

Enter the text to encrypt: hello
Enter the shift value (1-25): 3
Encrypted text: khoor
```
